Tutorial: Como importar o banco de dados no XAMPP (phpMyAdmin)

1. Abra o XAMPP
   Inicie os serviços Apache e MySQL.

2. Acesse o phpMyAdmin
   No navegador, digite: [http://localhost/phpmyadmin](http://localhost/phpmyadmin)

3. Crie o banco de dados
   Clique em “Novo” no menu lateral esquerdo.
   No campo “Nome do banco de dados”, digite: cadastro_ebook
   Clique em “Criar”.

4. Importe o arquivo SQL
   Após criar o banco, clique sobre ele no lado esquerdo.
   Vá até a aba “Importar” na parte superior.
   Clique em “Escolher arquivo” e selecione o arquivo: cadastro_ebook.sql (que está dentro da pasta do projeto).

5. Execute a importação
   Clique em “Executar” na parte inferior da tela.
   Se tudo estiver certo, aparecerá a mensagem: Importação executada com sucesso.

6. Verifique os dados
   Clique no banco cadastro_ebook.
   Clique na tabela leads.
   Verifique se as colunas id, nome, email e whatsapp aparecem.

7. Pronto
   O banco de dados está pronto para ser usado pelo projeto PHP.
   Agora, basta abrir o projeto pelo navegador com o endereço:
   [http://localhost/pasta_do_projeto/](http://localhost/pasta_do_projeto/)
